About the author, credits & links

This program has been made by Antoine Potten with Borland Delphi 7.

It has been designed to run on Windows NT4 SP4, 2000, XP, Vista, 7, 8 and 10. It should still work fine on Windows 98 and ME too (except unicode support).

At first, I made this program because the only file renamers that I found were shareware (at that time).
