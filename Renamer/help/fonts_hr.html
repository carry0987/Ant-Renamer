Font koji se koristi za posebne znakove

U prozoru Opcije možete odabrati koji font treba koristiti na mjestima kod kojih se javljaju posebni znakovi. Ta opcija je jedino dostupna na NT sustavima (NT4, 2000, XP, Vista, 7, ...) dok Unicode nije podržan na Windows 95, 98, ME.

By default, the program tries to use one of these:

  • Noto Sans, which can be obtained from Google Fonts,
  • DejaVu Sans, which can be obtained from DejaVu fonts,
  • Arial Unicode MS, which came with older versions of Microsoft Office but was not included in default install (it is called "Universal font" in the installation options).

Ako koristite samo osnovne latiniène znakove, možete odabrati bilo koji font font (npr. "MS Sans Serif" koji je zadan za amerièke & zapadno-europske sustave). Ali ako posebni ili strani znak nije prikazan ispravno (npr. vidite kvadrate umjesto znakova), možda æete morati odabrati neki drugi font.

The "Arial" font included in Windows 2000 and later versions should work for all European (West & Central) languages, Cyrillic, Greek, Hebrew and Arabic.

For old versions of Windows, here is a list of the font that can be used for some Asiatic langages, if they are installed on your system (those were used by Internet Explorer and Mozilla for displaying web pages that are written in these langages).

Pojednostavljeni kineski:SimSun, NSimSun
Korejski:GulimChe
