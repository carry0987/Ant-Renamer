சரங்களை வடிவமைத்தல்

சில செயல்களில், "%" எழுத்துக்குறிகளால் வரையறுக்கப்பட்ட சிறப்பு குறிச்சொற்களைப் பயன்படுத்தலாம்.

"%" பின்னர் ஒரு சிறப்பு எழுத்துக்குறியாகப் பயன்படுத்தப்படுவதால், கோப்பு பெயரில் எளிய "%" ஐ வைத்திருந்தால், நீங்கள் "%%" ஐ உள்ளிட வேண்டும்.

சிறப்பு குறிச்சொற்களை ஆதரிக்கும் ஒவ்வொரு செயலிலும், கிடைக்கக்கூடிய குறிச்சொற்கள் இந்த செயலின் அளவுருக்களின் கீழ் "குறிப்புகள்" உரையில் பட்டியலிடப்பட்டுள்ளன.

எடுத்துக்காட்டாக, % name% மற்றும் %ext% ஆகியவற்றுடன் %author% மற்றும் %title% போன்ற ஐடி3-தொடர்பான குறிச்சொற்களை எம்பி3 செயலைப் பயன்படுத்து ஆதரிக்கிறது. அதனால் ஒரு எம்பி 3 கோப்புகளுக்கான வழக்கமான முகமூடி இப்படி இருக்கும் %author% - %title%%ext% (%ext% புள்ளியை உள்ளடக்கியது, எனவே நாம் புள்ளியை முகமூடியில் வைக்க தேவையில்லை இங்கே பயன்படுத்தப்படுகிறது).

பொதுவாக %folderN% குறிக்குழு கிடைக்கும். இந்த ஒட்டு, தாய்க் கோப்புறைகளின் பெயரைக் கோப்புப் பெயரிலேயே பயன்படுத்திக் கொள்ள உதவுகிறது. இதை விளக்குவது சற்று கடினம், எனவே நான் உங்களுக்கு சில உதாரணங்கள் தருகிறேன்.

உங்களிடம் "இ:\தகவல்\படங்கள்\திரைப்படங்கள் சுவரொட்டி.செபிசி\" இல் கோப்பு இருந்தால், நீங்கள் சரம் செருகும் செயலைப்

  • முகமூடி = முகமூடி %folder1% -  கோப்பு "திரைப்படங்கள் - சுவரொட்டி.செபிசி" என மறுபெயரிடப்படும்.
  • முகமூடி = முகமூடி %folder2% -  கோப்பு "படங்கள் - சுவரொட்டி.செபிசி" என மறுபெயரிடப்படும்
  • முதலியன.

நீங்கள் ஒரு உறவினர் பாதையையும் பயன்படுத்தலாம், எ.கா. உங்களிடம் பல கோப்புறைகள் இருந்தால், ஒவ்வொன்றிலும் நிறைய கோப்புகள் உள்ளன, மேலும் இந்த கோப்புகள் அனைத்தும் குறிப்பாக தனித்துவமானவை அல்ல இரண்டு வெவ்வேறு கோப்புறைகள். கோப்புறை பெயரை நுழைக்க விரும்பலாம், பின்னர் அனைத்து கோப்புகளையும் பொதுவான தாய் கோப்புறைக்கு நகர்த்தலாம். இங்கே ஒரு உதாரணம்; உங்களிடம் பின்வரும் கோப்புகள் உள்ளன:

  • இ: \ தரவு \ படங்கள் \ திரைப்படங்கள் \சுவரொட்டி1.செபிசி
  • இ: \ தரவு \ படங்கள் \ விளையாட்டுகள் \சுவரொட்டி1.செபிசி
  • இ: \ தரவு \ படங்கள் \ விளையாட்டுகள் \சுவரொட்டி2.செபிசி

நீங்கள் அவற்றை மறுபெயரிட்டால், சரம் செருகும் செயல் மற்றும் ஒரு சரம் போன்ற ../%folder1%  தொடக்கத்திலிருந்தே நிலை 0 இல் செருகப்பட்டால், நீங்கள் பின்வருவனவற்றைப் பெறுவீர்கள் "படங்கள்" கோப்புறையில்:

  • திரைப்படங்கள் சுவரொட்டி1.செபிசி
  • விளையாட்டுகள் சுவரொட்டி1.செபிசி
  • விளையாட்டுகள் சுவரொட்டி2.செபிசி

இப்போது நீங்கள் வெற்று "திரைப்படங்கள்" & "விளையாட்டுகள்" கோப்புறைகளை நீக்கலாம்.

அதற்கு பதிலாக நீங்கள் எதிர்மறை எண்ணை உள்ளிட்டால், எண்ணிக்கை தற்போதைய கோப்புறையிலிருந்து அல்லாமல் வேரிலிருந்து செய்யப்படும். எனவே இன்னும் "இ:\தரவு \ படங்கள் \ திரைப்படங்கள்\" ஐப் பயன்படுத்துகிறது கோப்புறையாக:

  • %folder-1% = "இ"
  • %folder-2% = "தரவு"
  • %folder-3% = "படங்கள்"
  • முதலியன.

%name% குறிச்சொல்லைப் பயன்படுத்தி, கோப்புகளை கோப்பின் பெயரை எடுக்கும் கோப்புறையில் வைக்கவும் நீங்கள் விரும்பலாம். அதற்காக, "கோப்புறைகள் கட்டாயமாக்கு" விருப்பம் இயக்கப்பட வேண்டும். உங்களிடம் பின்வரும் கோப்புகள் இருந்தால்:

  • மகிழூந்துகள்.டிஒசி
  • மகிழூந்துகள்.எஃஎல்எச்
  • மகிழூந்துகள்.செபிசி
  • வீடுகள்.டிஒசி
  • வீடுகள்.எஃஎல்எச்
  • வீடுகள்.செபிசி

இன்னும் சரம் செருகும் செயலுடன், போன்ற ஒரு சரத்தைப் பயன்படுத்தி %name%\ ஆரம்பத்தில் இருந்து நிலை 0 இல் செருகப்பட்டால், நீங்கள் மூன்று "மகிழூந்துகள்" கோப்புகளைப் பெறுவீர்கள் ஒரு "மகிழூந்துகள்" கோப்புறை, மற்றும் "வீடுகள்" கோப்புறையில் மூன்று "வீடுகள்" கோப்புகள்.

கோப்புறைகளை மறுபெயரிடும் போது, %count% மற்றும் சில வழித்தோன்றல் குறிச்சொற்கள் கிடைக்கின்றன:

  • %count% = மறுபெயரிடப்பட்ட கோப்புறையில் உள்ள கோப்புகள் மற்றும் துணை கோப்புறைகளின் எண்ணிக்கை
  • %countr% = அதே ஆனால் துணை கோப்புறைகளில் உள்ளதையும் கணக்கிடுகிறது (மீள்நிகழ்வு)
  • %countfiles% = கோப்புகளை மட்டுமே எண்ணுகிறது
  • %countfilesr% = அதே ஆனால் துணை கோப்புறைகளின் கோப்புகளை உள்ளடக்கியது (மீளுருவாக்கம்)
  • %countfolders% = துணை கோப்புறைகளை மட்டுமே எண்ணுகிறது
  • %countfoldersr% = அதே ஆனால் துணை கோப்புறைகளின் துணை கோப்புறைகளை உள்ளடக்கியது (மீளும்)

பயன்படுத்த எம்பி3 குறிச்சொற்கள் மற்றும் பயன்படுத்த கோப்பக தகவல் செயல்கள், ஒரு சிறப்பு குறிச்சொல் உள்ளது: $if. நிபந்தனை வடிவமைப்பை உருவாக்க இது உங்களை அனுமதிக்கிறது. உதாரணத்திற்கு, எப்பொழுது ஒரு புலம் காலியாக உள்ளது, அதைத் தொடர்ந்து வரும் " - " ஐ நீங்கள் காண்பிக்க விரும்பாமல் இருக்கலாம். அல்லது ஒரு சிறப்பு மதிப்பு வைக்க.

தொடரியல் பின்வருமாறு: $if(நிபந்தனை, ஆம்-எனில்-மதிப்பு, இல்-எனில்-மதிப்பு). "நிபந்தனை" ஒரு வெற்று சரம் இல்லை என்றால், ஆம்-எனில்-மதிப்பு பயன்படுத்தப்படும். மற்றொன்றில், இல்-எனில்-மதிப்பு பயன்படுத்தப்படும். $if செயல்பாட்டிற்குள் நீங்கள் காற்புள்ளிகள் மற்றும் அடைப்புக்குறிகளைப் பயன்படுத்த வேண்டியிருப்பதால், நீங்கள் மதிப்புகளை மேற்கோள் குறிகளுக்குள் வைக்கலாம் ("... ") செய்ய அவற்றை பகுப்பாய்வு செய்வதைத் தடுக்கவும். ஆனால் அவர்களுடன் கவனமாக இருங்கள்: அவை %tag% பகுப்பாய்வு செய்வதைத் தடுக்கின்றன.

அதை சிறப்பாக விளக்கும் சில எடுத்துக்காட்டுகள் இங்கே. கோப்புகள் பட்டியலில் இரண்டு எம்பி 3 கோப்புகள் உள்ளன, அவை கெட்ட பெயரைக் கொண்டுள்ளன (1.எம்பி3 மற்றும் 2.எம்பி3) ஆனால் அது பின்வருவனவற்றைக் கொண்டுள்ளது சரியான ஐடி3 தகவல்:

  • 1.எம்பி3: ஆசிரியர் = மங்கலாக, தலைப்பு = பாடல் 2, ஆல்பம் காலி, தடம் காலியாக உள்ளது
  • 2.எம்பி3: ஆசிரியர் = விவால்டி, தலைப்பு காலியாக உள்ளது, ஆல்பம் = லு குவாட்ரோ ச்டாகியோனி, தடம் = 1

பல்வேறு முகமூடிகளைப் பயன்படுத்தி முடிவு இங்கே:

%author% - %title%%ext%
வெறுமனே ஆசிரியர் மற்றும் தலைப்பை எடுத்துக் கொள்ளுங்கள், ஒரு கோடு மூலம் பிரிக்கவும். இங்கே நாம் if செயல்பாட்டை பயன்படுத்தவில்லை, மேலும் வெற்று புலன்கள் இருக்கும்போது இதன் விளைவாக நன்றாக இருக்காது.

  • மங்கல் - பாடல் 2.எம்பி3
  • விவால்டி - .எம்பி3
