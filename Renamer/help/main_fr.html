Fenêtre principale

La fenêtre principale de Ant Renamer est divisée en trois pages :

  • Fichiers : Cette page contient la liste des fichiers qui seront renommés.
  • Opérations : Cette page permet de choisir ce qui doit être fait avec les fichiers.
  • Journal : Cette page contient la liste de ce qui a été fait (ou n'a pas pu être fait) au fichiers.

Ces pages sont accessibles via la barre d'outils "Pages" qui ressemble à ceci :

Une fois que vous avez défini les fichiers à renommer et comment les renommer, cliquez le bouton "Démarrer". Le journal indique ce qui est arrivé à chaque fichier.

Le bouton "Annuler" restaure les anciens noms. Il n'y a qu'un niveau d'annulation, donc vous ne pouvez annuler que la dernière opération effectuée.

Le bouton "Toujours en avant-plan", lorsqu'il est actif, permet de conserver la fenêtre au-dessus autres fenêtres lorsqu'elle n'est pas active.

Le bouton "Options" ouvre la fenêtre des options.
